云同步设置

配置 Firebase Realtime Database 后，管理员修改的数据会自动同步到云端。
其他人员打开此网页即可实时看到最新的任务分配情况。

创建 Firebase 项目

访问 Firebase 控制台，点击"添加项目"，创建一个新项目（免费，无需绑卡）。

创建 Realtime Database

在左侧菜单找到"Realtime Database"，点击"创建数据库"。位置选亚洲，规则选择"测试模式"（适合内部使用）。

复制数据库网址

在 Realtime Database 页面顶部，复制网址。格式如：
https://xxxxx-default-rtdb.firebaseio.com
粘贴到上方输入框即可。
